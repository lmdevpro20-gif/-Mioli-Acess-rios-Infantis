## ⚡ 3 PASSOS PARA COMEÇAR

### 📍 Passo 1: Configurar Supabase (10 min)

1. Abra **`GETTING_STARTED.md`**
2. Siga o **Passo 1** completo
3. Copie as **credenciais** para `.env.local`

✅ Quando terminar: seu banco de dados está pronto

### 📍 Passo 2: Rodar Localmente (2 min)

```bash
npm install
npm run dev
```

✅ Quando terminar: aplicação rodando em `http://localhost:3000`

### 📍 Passo 3: Primeiro Login (1 min)

1. Abra http://localhost:3000
2. Faça login com o email criado no Supabase
3. Você chega no Dashboard 🎉

✅ Quando terminar: você está usando o sistema!
